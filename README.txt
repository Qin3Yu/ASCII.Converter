- 使用快捷转换转换数字时请使用主键盘的数字键输入。
- 使用硬转换转换之前请确认至少一个 textBox 为空。
- 硬转换可以转换任何可打印字符，但不可转换如 shift 、Esc 等特殊键。

- When using the quick conversion feature, please input the numbers using the main keyboard's numeric keys.
- Before using the hard conversion, ensure that at least one textBox is empty.
- The hard conversion can process any printable characters but cannot convert special keys such as Shift, Esc, etc.
